## Usage

```typ
#show: vtzone.fix-cjk-punct-vertical // Optional show rule
// Real content
#vtzone.vtzone(
  doc,
  x-scale: 100%,
  max-height: auto,
  horizontal: rtl,
  row-gutter: 0.37em,
  col-gutter: 0.5em,
  custom-parbreak: none,
  initial-skip: 0mm,
)[...]
```
